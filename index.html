Lessons (interactive)

Lesson 1 — What is AI?

Artificial Intelligence (AI) is a set of techniques that enable machines to mimic intelligence. Machine Learning (ML) is a subset of AI that learns from data. Deep Learning (DL) uses neural networks with many layers.

Key terms
  • Model: A function that maps inputs to outputs.
  • Training: The process of finding model parameters from data.
  • Overfitting: When a model learns noise and fails to generalize.

Lesson 2 — Data & Preprocessing

Good data is the backbone of ML. Learn to clean, normalize, handle missing values, and engineer features.

Example: scale values to zero mean and unit variance (z-score)

Lesson 3 — A simple model by hand

We'll implement linear regression in the Playground using TensorFlow.js. Open the Playground to run it live.

Lesson 4 — Evaluation & Metrics

Common metrics: accuracy, precision, recall, F1-score for classification; MSE and MAE for regression. Always evaluate on a test set you didn't train on.

Lesson 5 — Ethics & Bias

AI models can amplify bias in data. Think about fairness, privacy, and explainability. Regularly validate models across subgroups and keep humans in the loop.
